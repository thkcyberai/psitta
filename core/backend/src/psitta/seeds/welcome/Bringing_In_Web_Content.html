Web Content in Psitta

Have a blog post, an article, or a page you saved from the web? Bring it into Psitta as an HTML file and put it to work.

When you upload an HTML file, Psitta reads the text inside it — the words, not the code — and turns it into a document you can listen to and edit. Press Read to hear the article narrated. Open it in the Writing Desk to revise, quote, or build on it.

One honest note: HTML is an import format here, not an export target. Once it’s in Psitta, it becomes a Psitta document — editing it doesn’t write a new web page; it shapes a new piece of your own. That’s by design. HTML is the doorway in; your writing is what happens after.

So use HTML to gather raw material — research, references, a draft you wrote elsewhere — and let Psitta turn it into something you can hear and make your own.
