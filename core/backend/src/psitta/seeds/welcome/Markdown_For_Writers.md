# Markdown in Psitta

Markdown is plain text with a light touch of structure. You write `# like this` for a heading and `**like this**` for bold, and it just works — no toolbars, no fuss. Bloggers, note-takers, and technical writers love it because it keeps you in the flow of words while still giving your work shape.

## Why writers reach for it

- It's fast — your hands never leave the keyboard.
- It's portable — a Markdown file opens anywhere, forever.
- It's structured — headings, lists, and emphasis come for free.

## How it works in Psitta

This file is a Markdown document (.md). In Psitta it's a full **Write** format: your headings and emphasis round-trip cleanly, and you can edit it in the Writing Desk like any document. Press **Read** and Psitta narrates the words — cleanly, without speaking the `#` and `*` symbols aloud.

> Tip: Markdown is a wonderful way to draft. Write quickly in plain text, let the structure emerge, and export to a polished document when you're done.

Try changing a heading or adding a list above, then listen to how it reads. Markdown keeps the writing simple so the *thinking* can be the hard part.
